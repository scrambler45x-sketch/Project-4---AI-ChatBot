# 🤖 AI Chatbot using Flask & NLTK

A modern AI chatbot built with **Python**, **Flask**, **HTML**, **CSS**, **JavaScript**, and **NLTK**. This project provides a clean chat interface where users can interact with a chatbot capable of answering basic questions, greeting users, providing the current date and time, and demonstrating Natural Language Processing (NLP) concepts.

---

## ✨ Features

- Modern Chat Interface
- Real-time Messaging
- NLP using NLTK
- Greeting Detection
- Current Date Response
- Current Time Response
- Basic AI Question Answering
- Responsive Design
- Clean Dark Theme
- Easy to Extend
